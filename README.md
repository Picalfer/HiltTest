# Hilt Dependency Test

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)
[![Hilt](https://img.shields.io/badge/Dagger-Hilt-purple.svg)](https://dagger.dev/hilt/)

Тестовое приложение для изучения Dependency Injection с Hilt в Android. Демонстрирует различные scope и внедрение зависимостей в Activity, Fragment и ViewModel.

## Функциональность

- Внедрение зависимостей в Activity, Fragment и ViewModel
- Использование ActivityRetainedScope для сохранения состояния
- Навигация между фрагментами с общими зависимостями
- Демонстрация работы разных scope Hilt

## Технологии

- Kotlin
- Dagger Hilt
- ViewModel
- Navigation Component
- ActivityRetainedComponent
